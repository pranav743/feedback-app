## Getting Started

To run the application, open **two separate terminals** and execute the following commands:

### 1. Start the Backend

```bash
chmod +x start-backend.sh
./start-backend.sh
```

### 2. Start the Frontend

```bash
chmod +x start-frontend.sh
./start-frontend.sh
```
